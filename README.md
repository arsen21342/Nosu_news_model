Обучение генеративной языковой модели на осетинском языке.

Целью работы является создание демонстрационной модели, которая может генерировать новости на осетинском языке.

Основные этапы работы:
- Сбор и подготовка осетинских текстов (на данный момент новости с осетинской версии официального сайта СОГУ)
- Обучение BPE-токенизатора
- Обучение GPT-модели на небольшом датасете
- Тестирование генерации текста

Структура проекта:
- parser.py - скрипт, который парсит статьи с новостной колонки сайта oset.nosu.ru и сохраняет в файл ossetian_news_corpus.txt.
- to_jsonl.py - скрипт который формирует jsonl-файл на основе ossetian_news_corpus.txt. По итогу формируется ossetian_corpus.jsonl, который будет использоваться в обучении модели в качестве датасета.
- train_tokenizer.py - обучение BPE-токенизатора на основе jsonl-файла.
- train.py - скрипт обучения модели. После обучения формируется файл модели ossetian_gpt_model.pth.
- test.py - скрипт для проверки результата обучения.

После установки проекта и всех зависимостей из файла requirements.txt можно последовательно запустить скрипты обучения токенизатора (train_tokenizer.py), затем обучение модели (train.py), и после этого тестировать модель в test.py. Но в проекте есть уже есть
обученная модель с токенизатором, благодаря чему можно сразу преступить к проверке модели в test.py.

При запуске тестирования вы увидите меню с тремя режимами:
1. Быстрое тестирование – запуск на заранее заданных промптах.
2. Интерактивный режим – вводите свой текст и получаете продолжение.
3. Сравнение температур – генерация одного промпта с разными настройками креативности.

Пример:

Введите промпт: ЦИПУ

ЦИПУ - йы студент – егъау телеуынадон равдысты архайæг . Джызойты Ацæмæз , ЦИПУ - йы психологон - педагогон факультеты 2 - æм курсы магистрант , сси СТС - ы телеуынадон равдыст « Хъæбатырты хъæзтытæ »- йы архайæг .
Равдысты хъайтартæ – 18 азæй 70 азмæ кæуыл цæуы , нæ бæстæйы

Модель обучена на корпусе объёмом чуть больше 300 новостных статей на осетинском языке. Она способна генерировать связные тексты на осетинском языке, повторяющие структуру и стиль новостей.

Это демонстрационный прототип, который показывает принципиальную возможность создания LLM для осетинского языка с ограниченными вычислительными ресурсами. Для более лучшей реализации модели потребуется найти и обработать больше информации, текущего датасета 
достаточно лишь для демонстрации формирования новостей СОГУ.
